# DTShoppingCart

# Pre-requisites
Please make sure the following are installed on your machine:
* .NET core 2
* Sql LocalDB

Following technologies have been used to implment the shopping cart discount solution:
* ASP.NET CORE 2
* EntityFramework Core 2
* SQL LocalDB

# Design Patterns
